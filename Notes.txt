C++ in Linux:
  To complie and run via CLI
    g++ program.cpp -o program      ---> this compiles program.cpp and names the executable file as program
    ./program                       ---> this runs the executable file program
  or
    g++ % -o program && ./program   ---> % means current file
  or
    :!g++ % -o program && ./program ---> Directly run this in vim, ! -> runs the command in shell
