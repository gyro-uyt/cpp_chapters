## What is venv?

Virtual environment is method to create isolate implementains of python libraries in differect projects.

## How to use?

Firstly, do this,  
`python -m venv <nameOfVenv>` 

then activate it,

* In Linux, fish shell by,  
`source env/bin/activate.fish`  

* In Linux, bash shell by,  
`source env/bin/activate`  

* In Windows, cmd by,  
`.\Scripts\activate`
