# macOS Sequoia Clone

This project is a clone of the macOS Sequoia operating system, built with HTML, CSS, and JavaScript.

## Features

- **Window Management:** Open, close, minimize, and maximize windows.
- **Draggable Windows:** Move windows around the desktop by dragging the title bar.
- **Dock:** A functional dock with application icons that bounce on click.
- **Clock:** A real-time clock in the menu bar.
- **Applications:**
  - **Chrome:** A simulated web browser with an iframe to load websites.
  - **Word:** A basic text editor with formatting options.
  - **Calculator:** A simple calculator with basic arithmetic operations.
  - **Paint:** A drawing application with color and brush size options.

## Getting Started

To run this project, simply open the `index.html` file in your web browser.

## Project Structure

- `index.html`: The main HTML file.
- `css/style.css`: The stylesheet for the project.
- `js/script.js`: The JavaScript for the project.
- `assets/`: A directory for storing assets (not used in this version).
