# Shell

What is shell, bash, fish, terminal, console, promt, cmd, etc.

- Terminal short for terminal-emulator is a window/program that runs different shell languages like bash, fish, zsh, etc.

- Shell

- Shell -> Bash ->> fish,zsh,etc.

- Bash is the language that runs in most terminal-emulators.

- How to copy/paste/scroll in bash? is a stupid question, as it doesn't depends on the bash, these things depends on the terminal you're using.

- Command line is the line in which we are currently in.

- Prompts are built-in your shells, this "➜  ~ "(PS1) is a promt

- bash vs zsh vs fish
    - bash is default on almost all machines
    - zsh is very similar to bash but includes easy customization
    - fish customizations already built-in but it's not POSIX compliant, meaning it uses different syntax than Bash, it's syntax is different than bash.
